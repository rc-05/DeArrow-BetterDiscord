# DeArrow-BetterDiscord
DeArrow integration for BetterDiscord

## What is this?
This is a BetterDiscord plugin that seamlessly integrates
[DeArrow](https://dearrow.ajay.app/) into Discord for less "clickbaity" YouTube
titles.

## How do I install it?
Grab the `DeArrow.plugin.js` script file and place it inside BetterDiscord's
plugin folder then enable it.

Or download the latest release and unzip the `DeArrow.plugin.js` file into
the BetterDiscord's plugin folder.
